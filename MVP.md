# CALIBRATED MVP Spec

**Target Release:** Early July 2026

## Core Game Modes

- **Competitive**: 2-minute timed range session
- **Accuracy Drills**: Focused skill practice
- **Practice Range**: Free-play warm-up

## Mechanics

- **Custom 9mm Pistol** with realistic recoil pattern (CS2-inspired)
- **Accuracy Cone System** for tracking precision
- **Hybrid Scoring**: Time-based + accuracy-based (center mass + headshot bonuses)
- **Target Types**: Traditional silhouettes + pop-up range

## Daily Quest Hook

"Complete 10 minutes" =
- 2x Competitive (4 min ea)
- 2x Accuracy Drills (4 min ea)
- 1x Practice (2 min)

## Monetization

- **Dual Currency**: In-game + premium
- **Gun Skins**: Cosmetic only
- **Seasonal Charms**: Cosmetic only
- **Range Decorations**: Cosmetic only (no gameplay impact)

## Explicitly NOT in MVP

- ✅ QTE/combo systems (feature creep)
- ✅ Active reload mechanics (post-launch)
- ✅ Seasonal rotating drills (framework ready, execution post-MVP)

## Status

Awaiting Roblox masterclass completion before resuming dev work.
